# similar-music
A music similarity library, returning a weighted combination of several API and tool results.

## Installation

```bash
npm install similar-music
```

## Usage

For now, see [the tests](test/index.js).

## Features

### `similarArtists`

Backends:
- Pandora
- Last.fm
- Spotify

### `similarTracks`

Backends:
- Last.fm
